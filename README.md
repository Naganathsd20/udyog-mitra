# 🚀 UdyogMitra - Full Stack MERN Job Portal

UdyogMitra is a modern Full Stack Job Portal developed using the MERN Stack. It provides a seamless platform where students can search and apply for jobs, while recruiters can manage companies, post job opportunities, and review applications. The application is fully responsive and deployed on the cloud.

---

## 🌐 Live Demo

🔗 https://udyog-mitra.vercel.app/

# ✨ Features

## 👨‍🎓 Student Module

- User Registration & Login
- Secure JWT Authentication
- Browse & Search Jobs
- View Job Details
- Apply for Jobs
- Save/Bookmark Jobs
- Upload Resume
- Profile Management
- View Applied Jobs
- Responsive UI

---

## 🏢 Recruiter Module

- Recruiter Registration & Login
- Company Registration
- Company Management
- Upload Company Logo
- Create Job Posts
- Manage Posted Jobs
- View Applicants
- Accept/Reject Applications
- Recruiter Dashboard

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/Naganathsd20/udyog-mitra.git
```

---

## Install Frontend

```bash
cd frontend
npm install
npm run dev
```

---

## Install Backend

```bash
cd backend
npm install
npm run dev
```

---

# 🔐 Environment Variables

Create a `.env` file inside the backend folder.

```env
PORT=8000

MONGO_URI=your_mongodb_connection_string

SECRET_KEY=your_secret_key

CLOUDINARY_CLOUD_NAME=your_cloud_name


```
